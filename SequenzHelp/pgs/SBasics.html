Sequenz Basics

Sequenz is a simple and easy to use webcam application. It takes pictures with your iSight camera with a selectable time interval and uploads them to your FTP server. It supports secure FTP connections over TLS/SSL.

To get started, fill out the FTP server values, like server address, user name, password and path on the server. You can set the default image filename to any name you want. It will be uploaded with an extension ".jpg" in jpeg file format. The file dimension is 640x480 pixel.

Sequenz is freeware. If you like Sequenz and want to support its future development, please donate via PayPal.
